# Sistema de Gestión de Libros - README

## 📚 Descripción
Aplicación web para la gestión de libros donde autores pueden publicar sus obras y lectores pueden descubrir contenido literario. Desarrollado con Laravel 10 y Tailwind CSS.

## ✨ Características
- **Para Autores**:
  - Crear y gestionar libros
  - Gestionar perfil

- **Para Lectores**:
  - Explorar catálogo
  - Leer libros online
  - Filtrar por Generos




## 🛠 Tecnologías
| Área       | Tecnologías                     |
|------------|---------------------------------|
| Backend    | Laravel 10, PHP 8.1+            |
| Frontend   | Blade, Tailwind CSS, Alpine.js  |
| Base datos | MySQL 8                         |
| Otros      | Git Flow, Docker (opcional)     |
